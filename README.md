# sasha-math-bot-2 — Bot 2.0 (dev-форк)

Форк [sasha-math-bot](https://github.com/billyrubinius-crypto/sasha-math-bot) для разработки
геймификации (бублики, сезоны, магазин, достижения, лиги, карта тем) без риска для боевого
проекта. Прод не трогается никогда — у форка свой Supabase-проект (`sasha-math-bot-2-dev`) и
свой тестовый Telegram-бот.

Контекст, спецификация этапа 1 и карточки задач — в приватном репозитории
`sasha-math-bot-private`, папка `bot2/`:

- `bot2/BOT2_CONTEXT.md` — архитектура, карта функций, правила
- `bot2/SPEC_STAGE1.md` — точные числа экономики этапа 1
- `bot2/seed.sql` — сид тестовых данных для dev-БД
- `bot2/tasks/` — карточки задач

Готовые деплойменты приходят из этого репозитория (GitHub Pages для `index.html`/
`teacher.html`; боты — отдельным хостингом при необходимости).
